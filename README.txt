Πακέτο: iliako_systima
Περιεχόμενο:
- index.html (το παιχνίδι)
- background.jpg (το φόντο)    -> αντικατάστησε με δικό σου αν θέλεις
- sun.gif                       -> ο Ήλιος (εικόνα που παρέχεις)
- planets/planet1.gif ... planet8.gif  -> βάλτε εδώ τα GIFs των πλανητών σας
- sounds/bravo1.wav, bravo2.wav, bravo3.wav -> προσωρινοί ήχοι επιβράβευσης

Οδηγίες:
1. Βάλτε τα δικά σας planet GIFs στο φάκελο 'planets' ονομάζοντάς τα planet1.gif ... planet8.gif
2. Αντικαταστήστε background.jpg ή sun.gif αν θέλετε άλλη εικόνα
3. Ανοίξτε index.html τοπικά για δοκιμή, ή ανεβάστε όλο το φάκελο σε GitHub Pages / Itch.io / hosting.
4. Στο Genially χρησιμοποιήστε Insert -> Other -> HTML iframe και βάλτε το URL προς το index.html (π.χ. GitHub Pages).
